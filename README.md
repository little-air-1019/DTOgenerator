# DTOgenerator
Spring Boot DTO產生器

## Features
- Beautify JSON input for easier editing.
- Select Validation API (Java EE vs Jakarta EE) to switch validation imports between `javax.*` and `jakarta.*` packages.
  - Java EE (javax) - For Spring Boot 2.x and earlier
  - Jakarta EE (jakarta) - For Spring Boot 3.x and later
- Note: This is independent of Java language version (8, 11, 17, 21, etc.)
